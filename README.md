# Lane-Line-Detection-Using-YOLOv5
This project introduces a deep learning system for real-time lane and road surface detection using CNNs. It delivers high accuracy across varied lighting and weather conditions, outperforming existing methods in speed and reliability, supporting advances in autonomous driving

1. **Project Goal:** The system aims to detect lane markings and road surfaces accurately in real time.
2. **Technology Used:** It is built using convolutional neural networks (CNNs), a powerful deep learning architecture.
3. **Performance:** The model maintains high accuracy in different lighting and weather conditions, ensuring reliable operation.
4. **Dataset:** Training and validation were done on a diverse dataset to improve generalization.
5. **Comparison:** It shows better detection speed and reliability compared to existing approaches.
6. **Contribution:** The work supports broader advancements in autonomous driving technologies.
